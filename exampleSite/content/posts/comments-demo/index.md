## Default Setting

By default, Pocket Hugo theme enables comments on all pages. If you wish to disable comments on a specific page or post, please add `comments: false` to the frontmatter section of the corresponding Markdown file.


This theme uses Twikoo as its default comment system. To utilize Twikoo, you must deploy the necessary cloud functions in accordance with the official Twikoo documentation to ensure proper operation. Alternatively, you are free to configure your preferred comment solution, such as Utterances, Waline, Disqus, and others, according to your own preferences.

- [Twikoo Docs](https://twikoo.js.org/en/intro.html)

- [Giscus Docs](https://giscus.app/)

- [Utterances Docs](https://utteranc.es/)

- [Waline Docs](https://waline.js.org/)

- [Disqus Docs](https://disqus.com/)
