## Installation

```bash
git submodule add https://github.com/h2dcc/pocket-hugo-theme.git themes/pocket-hugo-theme
```

```toml
theme = "pocket-hugo-theme"
```

## Aperçu local

```bash
hugo server --source exampleSite --themesDir ../..
```

## Points à vérifier

- La densité des cartes avec couverture
- Le rythme de lecture sur les pages d article
- Le comportement du sélecteur de langue
- L ordre des widgets de la colonne latérale
