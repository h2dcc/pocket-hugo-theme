Pocket Notes is a bilingual demo site for testing the Pocket Hugo Theme.

## What this site covers

- multilingual bundles
- single-language bundles
- flat Markdown posts
- archive, links, search, and talks pages

This setup is intentionally small, but it is varied enough to expose layout issues quickly.
